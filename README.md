# Breathing Room – Conference Space Rental Platform

A modern platform for searching, listing, and inquiring about conference spaces.

## Tech Stack
- React (frontend)
- Node.js + Express.js (backend)
- MongoDB (database)

## Features
- User authentication (JWT, OAuth placeholder)
- Admin dashboard
- Search & filter for venues
- Listing management for venue owners
- Booking and inquiry system

## Getting Started
1. `npm install`
2. `npm start`

## Folder Structure
- `src/components` – Reusable UI components
- `src/pages` – Main pages (Login, Register, Dashboard, Listings, etc.)
- `src/routes` – Route definitions and guards
- `src/context` – Auth and user context
- `src/utils` – Utility functions (API, helpers)
- `src/assets` – Images, icons, etc.

---
This is the frontend. Backend and database are separate.
